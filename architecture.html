Deployment Architecture

  • Cloud-Based Systems (70%): IngeniousMed, AdvancedMD, McKesson Clearinghouse
  • On-Premise Systems (20%): Meditech (Hospital data centers)
  • Manual Processes (10%): Bank reconciliation, Excel tracking, Portal access

Site-Specific Deployments

  • Each hospital maintains its own Meditech instance
  • Census data pulled individually from each facility
  • Some sites use direct Meditech reports
  • One site provides census via secure messaging
  • All consolidated in cloud-based IngeniousMed
